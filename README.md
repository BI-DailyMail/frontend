## 📌 서비스 소개

**매일메일**은 AI를 활용해 이메일의 보안 위협과 다크 데이터를 자동 탐지하고, 핵심 내용을 요약해주는 스마트 이메일 보안 에이전트입니다.

### 핵심 기능

| 기능 | 설명 |
|------|------|
| **AI 메일 분석** | Gemini AI가 메일 내용을 요약·보안 등급 분류·다크 데이터 감지 |
| **보안 센터** | 위험 / 주의 / 안전 등급별 메일 분류 및 상세 분석 이유 제공 |
| **다크 데이터 탐지** | 피싱·스팸·개인정보 노출 등 문제 메일 별도 관리 |
| **스마트 필터** | 사용자 정의 스팸 키워드 등록·활성화·비활성화 관리 |
| **광고 수익 모델** | 대시보드 하단 스폰서 광고 배너 (수수료 기반 수익 모델 시연) |

### 보안 등급 기준

| 등급 | 설명 |
|------|------|
| 🔴 위험 | 피싱·사기 등 즉각적인 금전·개인정보 피해 가능성 |
| 🟡 주의 | 스팸·과장 광고 등 클릭 시 위험할 수 있는 메일 |
| 🟢 안전 | 보안 위협 없음 (다크 데이터 포함 시 별도 표시) |

실제 Gmail 연동 없이 아래 샘플 데이터로 동작합니다.

### 메일 (6개)

| 제목 | 발신자 | 보안 등급 | 다크 데이터 |
|------|--------|-----------|-------------|
| [긴급] 계좌 이상 거래 감지 | 국민은행 보안팀 | 위험 | ✓ |
| [네이버] 로그인 차단 안내 | 네이버 계정보안센터 | 위험 | - |
| [광고] 직장인 대출 한도 UP! | 빠른대출 고객센터 | 주의 | ✓ |
| [광고] 건강기능식품 80% 할인 | 건강한하루 쇼핑몰 | 주의 | - |
| 계약서 초안 공유 | 이진수 | 안전 | ✓ |
| 6월 팀 회식 일정 안내 | 최민준 | 안전 | - |

### 스마트 필터 키워드 (기본 8개)

| 키워드 | 상태 |
|--------|------|

## ⚙️ 로컬 실행 방법

### 1. 패키지 설치

```bash
npm install
```

### 2. 환경 변수 설정

프로젝트 루트에 `.env.local` 파일 생성:

```
VITE_API_URL=http://localhost:8000
```

> 백엔드 서버가 없으면 메일 목록·스마트 필터는 Mock 데이터로 동작하며,
> 대시보드 AI 분석 기능은 백엔드(`localhost:8000`) 실행 시에만 동작합니다.

### 3. 개발 서버 실행

```bash
npm run dev
```
